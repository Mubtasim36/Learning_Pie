# PythonShikhi

[![Python](https://img.shields.io/badge/python-3.x-blue?style=for-the-badge)](https://www.python.org/)
[![Last Commit](https://img.shields.io/github/last-commit/Mubtasim36/Learning_Pie?style=for-the-badge)](https://github.com/Mubtasim36/Learning_Pie)

PythonShikhi is a beginner-friendly Python learning repository designed to help learners explore core programming concepts through simple, practical examples. The project currently focuses on fundamentals such as input/output, arithmetic, strings, lists, and conditionals.

---

## Contents

- [Basics](Basics/) — A complete beginner module covering input/output, arithmetic, strings, lists, conditionals, and simple practice tasks. **Status: Completed**
- Root-level examples — Standalone Python scripts for arithmetic, conditionals, and small practice programs. **Status: Active**

## Project Overview

This repository is organized to make learning Python approachable and structured. The Basics folder acts as the main learning path, while the root directory also contains additional example files that can be explored independently.

## What You Will Learn

- Writing and running Python scripts
- Using variables and input/output
- Performing arithmetic operations
- Working with strings and lists
- Applying conditional logic
- Building simple practice programs and mini exercises

## How to Use This Repository

1. Open the [Basics](Basics/) folder to begin with the structured lessons.
2. Read the [Basics README](Basics/README.md) for explanations and instructions.
3. Run the Python scripts from the terminal using Python 3.
4. Try modifying the examples to explore how Python behaves with different inputs.

## How to Run the Examples

From the Basics folder, run:

```bash
python main.py
```

Or on some systems:

```bash
python3 main.py
```
