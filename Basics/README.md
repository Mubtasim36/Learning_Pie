# Python Basics Learning Examples

## Overview
This folder contains a collection of beginner-friendly Python practice files designed to introduce core programming concepts. The examples cover input/output, basic arithmetic, strings, lists, conditionals, loops, and simple problem-solving exercises.

## Concepts Covered
The examples in this folder demonstrate:
- Printing output to the console
- Accepting user input
- Using variables and performing arithmetic operations
- Working with strings and string formatting
- Creating and manipulating lists
- Using conditional statements for decision-making
- Writing simple loops and practice programs
- Importing functions from other Python files

## Learning Objectives
By working through these examples, learners will be able to:
- Write and run simple Python scripts
- Understand how input and output work in Python
- Perform basic calculations using operators
- Work with strings and lists effectively
- Apply conditionals and simple loops in practical examples
- Organize code across multiple Python files

## File Descriptions
- main.py: The main entry point of the project. It displays a menu and lets the user choose which tutorial or practice module to run.
- helper.py: A small helper module that provides the reusable greet() function.
- Basics.py: Demonstrates basic string operations, concatenation, indexing, and slicing.
- basicCalc.py: Introduces simple arithmetic and basic calculations using user input.
- lists.py: Shows how to create, access, modify, and analyze lists, including nested lists and list aliasing.
- NumericConditionals.py: Covers numeric conditionals such as positive/negative checks, even/odd checks, leap year logic, prime number checks, grade evaluation, BMI logic, and quadrant detection.
- StringConditionals.py: Covers string-based conditionals such as lowercase/uppercase checks, password strength checks, vowel/consonant checks, palindrome detection, and input type detection.
- smallTasks.py: Contains a few small practice programs, including a Fibonacci series example and a simple login/register example.

## How to Run the Examples
1. Open a terminal or command prompt.
2. Navigate to the Basics folder.
3. Run the main script:
   - On Windows: python main.py
   - On Linux/macOS: python3 main.py
4. Choose a number from the menu to explore the different topics.

Suggested menu options:
- 1: Basics
- 2: Basic Calculation
- 3: Lists
- 4: Small Tasks
- 5: Numeric Conditionals
- 6: String Conditionals
- 7: Practice

## Expected Output
Running the program will display a welcome message, a menu, and then interactive prompts based on the option you select. The exact output depends on your input, but you can expect messages such as:

```text
Welcome to my Python Learning Playground
Choose Operation: 1. Basics 2. Basic Calculation 3. Lists 4. Small Tasks 5. Numeric Conditionals 6. String Conditionals 7. Practice
Enter which operation you want to perform: From 1-7:
```

After selecting an option, the console will show the relevant tutorial output, such as arithmetic results, string examples, list operations, or conditional checks.

## Prerequisites
- Python 3.x installed on your system
- A terminal or command prompt
- Basic familiarity with navigating folders in the command line

## Notes
- Some modules are written as functions and are executed through main.py.
- Make sure you run the program from the Basics directory so the imports resolve correctly.
- These files are intended for learning and experimentation, so feel free to modify the inputs and values to see different results.
